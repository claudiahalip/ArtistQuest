# Welcome to ArtistQuest!

### Node.js
- use `v20.11.0` node version

## Install NPM packages

```sh
npm i
```


## Development

Run the dev server:

```sh
npm run dev
```

Run test:
```sh 
npm run test
```

## Deployment

First, build your app for production:

```sh
npm run build
```

Then run the app in production mode:

```sh
npm start
```
